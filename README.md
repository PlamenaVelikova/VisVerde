# VisVerde

**1.Концепция на бранда**
**Мисия**
Мисията на Vis Verde е да създава красиви, устойчиви и хармонични зелени пространства, които подобряват качеството на живот и свързват хората с природата.

**Целева аудитория**
Проектът е насочен към:
- собственици на къщи и вили;
- хора, търсещи модерно озеленяване;
- клиенти с интерес към декоративни растения;
- любители на градинарството и екологичния начин на живот.

**Ключови думи**
- Озеленяване
- Градини
- Декоративни растения
- Онлайн магазин
- Природа
- Зелени пространства
- Поддръжка на градини

**2.Визия**
**Лого**
Логото на Vis Verde комбинира природни елементи и модерна визия.

**Цветова палитра**
Основните цветове в проекта са:
  Тъмнозелено — hsl(123, 46%, 34%)
  Светлозелено — #4CAF50
  Бяло — #FFFFFF
  Тъмносиво — #1c1c1
Цветовата схема е подбрана така, че да създава усещане за природа и спокойствие.

**Типография**
Използван е модерен и лесно четим sans-serif шрифт:
- Arial
- Dynamic font sizing чрез clamp()

**3.Уебсайт**
Модерен уебсайт за озеленяване, растения и градински услуги.
Проектът включва продуктов каталог, количка за пазаруване, търсачка, галерия, контактна форма и модерен UI дизайн.

**Структура на уебсайта**
Уебсайтът включва няколко основни страници:
- Начална страница
- Услуги
- Онлайн магазин с продуктов каталог
- За нас
- Контакти
Навигацията е организирана чрез responsive navbar с dropdown меню.

**Използвани технологии**
- HTML5
- CSS3
- JavaScript
- LocalStorage API
- AOS Animation Library
- jQuery
  
**Интерактивни елементи**
- Shopping cart система
- Търсачка за продукти
- Dropdown navigation
- Slideshow галерия
- Hover анимации
- Scroll animations
- Responsive design
- Dynamic filtering
- Quantity controls
- Cart modal overlay

**4.Мултимедия**
**Изображения**
Проектът използва:
- продуктови изображения;
- hero background снимки;
- slideshow галерия;
- икони за социални мрежи;
- декоративни изображения.

**Видео**
В страницата „За нас“ е добавено видео съдържание, което представя философията и услугите на бранда.

**5.Използване на изкуствен интелект**
Използвани инструменти:
- ChatGPT;
- GitHub Copilot;
- Synthesia.
  
Изкуственият интелект е използван при:
- генериране на видео;
- генериране на идеи за структура и дизайн;
- подобряване на UI/UX концепции;
- оптимизация на JavaScript функциите.
